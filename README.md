
# Orderly

Orderly helps you manage your restaurant easily. It takes control of the orders generated by customers and manages the availability of tables. Take your restaurant to the next level.


## Installation

Clone the repository
```bash
git clone https://github.com/m1randa26/Orderly.git

npm install

npm run dev
```
